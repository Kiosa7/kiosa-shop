# 🛒 Kiosa Shop

Proyecto personal de plataforma web para venta de productos y sistema de afiliados con generación de comisiones por enlaces compartidos.

---

## 📌 Descripción

**Kiosa Shop** es una aplicación web que permite:

- Visualizar productos digitales y físicos
- Generar links de afiliado
- Simular modelo de comisión por venta
- Explorar arquitectura de plataforma de afiliación
- Implementar flujo básico de e-commerce sin inventario

Este proyecto fue creado como práctica de desarrollo web y exploración de modelos de negocio digitales.

---

## 🎯 Objetivo del proyecto

- Practicar desarrollo frontend
- Implementar estructura de tienda online
- Simular sistema de afiliados
- Experimentar con modelo de ingresos por comisión
- Deploy en GitHub Pages
